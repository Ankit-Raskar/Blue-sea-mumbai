> Mumbai's most iconic luxury event venue on Worli Sea Face. 20+ years of crafting unforgettable weddings, corporate galas, and celebrations.

🔗 **Live Site:** [blue-sea-mumbai.vercel.app](https://blue-sea-mumbai.vercel.app/)

## 📌 About

**Blue Sea Mumbai** is a luxury banquet hall and event venue website located on the iconic Worli Sea Face, Mumbai. This project is a modern, responsive frontend website built to showcase the venue's offerings — from grand weddings and corporate events to private celebrations.

---

## ✨ Features

- 💍 Wedding & corporate event venue showcase
- 🌊 Sea-facing luxury venue details
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast loading with Vercel deployment
- 🎨 Elegant UI with smooth animations
- 📍 Location & contact information

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/Ankit-Raskar/Blue-sea-mumbai.git

# Navigate into the project
cd Blue-sea-mumbai

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

---

## 📦 Scripts

```bash
npm run dev       # Start development server
npm run build     # Build for production
npm run preview   # Preview production build
npm run lint      # Run ESLint
```
